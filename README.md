# Video Game Sales Prediction

A machine learning project that uses a Decision Tree Regressor to predict global video game sales based on features like platform, genre, year, and publisher.

## Dataset

Video Game Sales from Kaggle  
https://www.kaggle.com/datasets/gregorut/videogamesales

## Files

- modeling.ipynb - main notebook
- vgsales.csv - dataset
- requirements.txt - dependencies

## How to Run

Install dependencies:

pip install -r requirements.txt

Then open and run modeling.ipynb top to bottom.
